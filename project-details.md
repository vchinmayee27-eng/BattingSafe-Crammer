BattingSafe: Ultimate One-Night Exam Crammer & Study Workstation

Live Application URL: https://battingsafe-one-night-exam-crammer-1084634165024.asia-southeast1.run.app

Project Overview
BattingSafe is an all-in-one, high-yield academic workstation engineered specifically for students facing extreme, last-minute university exam timelines ("one-night batting"). Powered by Gemini via Google AI Studio's Build Mode, the application transforms dense, chaotic materials into structured, hyper-focused survival guides while integrating real-time productivity, gamification, and active recall utilities.

Advanced Features & Core Modules

1. Multi-Format Material Ingestion
Text & Syllabus Parsing: A large, responsive input area to process chaotic raw lecture notes.
Document Upload Support: Dedicated options allowing students to directly upload PDFs and PPTs to ingest entire lecture slide decks instantly.

2. High-Yield AI Extraction Dashboard
Core Concepts (ELIF5): Distills complex engineering theories into clear, simple, conversational explanations.
Vital Code & Formulas: Programmatically isolates essential programming syntax, core algorithmic blocks, and mathematical equations.
Predicted Exam Questions: Analyzes the weight of the input topics to generate high-probability university exam questions.

3. Active Learning & Cognitive Tools
Coding & Logic Memory Hacks: Generates custom mnemonics, visual analogies, and structural tricks to help students memorize intricate logic and syntax overnight.
Knowledge Map & Study Insights: Synthesizes the ingested syllabus into a clear, conceptual hierarchy map, offering intelligence on topic dependencies and structural weight.
Ask AI Doubt: A real-time, interactive chat utility allowing students to instantly cross-examine the material and resolve specific academic blockers on the spot.
Practice Quizzes: Automatically generates interactive mock questions to test retention and force active recall before entering the exam hall.

4. Integrated Productivity & Gamification Engine
Built-in Pomodoro Timer: A localized focus clock optimized for high-pressure cramming, pre-configured with standard productivity intervals (25-min study, 5-min chill, 15-min relax).
Streak Maintaining Option: Integrates a gamified daily retention tracker that rewards students for consecutive study days, building positive reinforcement and study momentum.

Technical Architecture
Deployment & Hosting: Deployed natively via Google AI Studio's cloud app infrastructure.
Core LLM Engine: Gemini, leveraged for multi-modal context processing (Text, PDF, PPT), zero-shot extraction, and interactive active-recall generation.
User Interface: A sleek, unified single-page dark-mode terminal optimized for high-speed scannability.
